Council of Salesforce — Agent templates

This folder contains per-role agent templates for the "Council of Salesforce". Each subfolder is a single agent and includes:
- `system.prompt.md` — system-level instruction (role/system prompt)
- `user.template.md` — user-facing prompt template
- `meta.json` — metadata (inputs, outputs, safety flags)

Add new agents by creating a new folder and following the same file structure.

Agents included:
- architect
- code-assistant
- test-engineer
- deployment-officer
- debugger
- compliance-auditor
- ux-flow-designer
